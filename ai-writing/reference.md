# AI writing tells — reference

Companion to [SKILL.md](SKILL.md). Distilled from [Wikipedia:Signs of AI writing](https://en.wikipedia.org/wiki/Wikipedia:Signs_of_AI_writing). Use when auditing a draft or expanding a checklist item.

## Content patterns

### Significance, legacy, broader trends

Watch for: stands/serves as, is a testament/reminder, crucial/pivotal/vital/significant/key role/moment, underscores/highlights its importance/significance, reflects broader, symbolizing its ongoing/enduring/lasting, contributing to the, setting the stage for, marking/shaping the, represents/marks a shift, key turning point, evolving landscape, focal point, indelible mark, deeply rooted, generated debate, prompted broader reflection, raising philosophical questions.

Rewrite move: delete the sentence, or replace with a specific fact (date, actor, measured outcome).

### Canned notability / media coverage

Watch for: independent coverage, local/regional/national media outlets, music/business/tech outlets, trade publications, profiled in, written by a leading expert, active social media presence, maintains a strong digital presence, significant/substantial/secondary coverage.

Rewrite move: cite a specific source for a specific claim; drop outlet-list peacocking.

### Superficial analysis endings

Watch for trailing -ing phrases: highlighting, underscoring, emphasizing, ensuring, reflecting, symbolizing, contributing to, cultivating, fostering, encompassing, enhancing; also "valuable insights", "align/resonate with".

Rewrite move: end at the fact; put interpretation in a separate sourced sentence if needed.

### Promotional language

Watch for: boasts a, vibrant, rich, profound, enhancing, showcasing, exemplifies, commitment to, natural beauty, nestled, in the heart of, groundbreaking, renowned, featuring, diverse array, seamless, cutting-edge, transformative.

Rewrite move: name the attribute without the marketing adjective.

### Vague attributions

Watch for: industry reports, observers have cited, experts argue, some critics argue, several sources/publications (when few are cited), "such as" before exhaustive lists.

Rewrite move: name the person, paper, or outlet; use singular if only one source.

### Challenges and future prospects formula

Watch for: Despite its… faces several challenges; Despite these challenges; Challenges and Legacy; Future Outlook; continues to thrive; positions them as critical; as [field] continues to evolve.

Rewrite move: list real constraints with evidence, or omit the section.

## AI vocabulary by era

Density matters: one word is weak evidence; a cluster is a strong tell. Do not ban synonyms that are not on the list.

### Common across eras

additionally (esp. sentence-initial), align with, boasts, bolstered, crucial, delve, emphasizing, enduring, enhance, fostering, garner, highlight (verb), interplay, intricate/intricacies, key (adj), landscape (abstract noun), meticulous/meticulously, pivotal, robust, showcase, tapestry (abstract), testament, underscore (verb), valuable, vibrant

### Prefer instead

| Avoid | Prefer |
|-------|--------|
| additionally | also / and / cut it |
| align with | match / fit / follow |
| boasts | has |
| bolster | support / strengthen |
| crucial / pivotal / key | important / main / (or cut) |
| delve into | look at / examine |
| emphasize / highlight / underscore | show / note / stress (sparingly) |
| enduring / testament | lasting / proof / sign |
| enhance | improve |
| foster | encourage / support / build |
| garner | get / receive |
| intricate | detailed / complex |
| landscape (abstract) | field / area / market |
| meticulous | careful / thorough |
| robust | strong / reliable |
| showcase | show / display |
| tapestry | mix / history / (rewrite) |
| utilize | use |
| vibrant | lively / active / (cut) |
| groundbreaking / cutting-edge | new / first / (say what is new) |

## Grammar and rhetoric

### Copula avoidance

Weak: serves as, stands as, marks, functions as, operates as, represents a, boasts, features, offers, maintains, refers to (when the subject is the thing, not the term).

Strong: is, are, has, have, does.

### Negative parallelisms

- not only X but also Y
- not just X — it's Y
- doesn't just X; it Y
- It's not X, it's Y
- no X, no Y, just Z
- rather than (as fake contrast)

### Rule of three

Adjective triples; "A, B, and C" when only one item is real; canned SEO-style listicles in threes.

### Elegant variation

Cycling "constraint / challenging climate / confines / state-imposed norms" for the same idea. Repeat the word instead.

## Style and formatting

- Title Case Headings → sentence case
- Overuse of bold for "key takeaways" → plain prose
- Inline-header lists (`- **Label**: text`) → prose or plain bullets
- Em dashes used as default punctuation → comma/period/paren
- Emoji section headers / emoji bullets → remove
- Curly quotes/apostrophes in code or markdown sources → straight quotes
- Markdown pasted into wikitext contexts → convert or remove
- Thematic breaks (`---`) before every heading → remove
- Unnecessary tiny tables for a few facts → prose

## Chatbot leftovers (always cut)

- Certainly! / Of course! / Absolutely! / Great question!
- I'd be happy to help
- I hope this helps!
- Let me know if you need anything else
- Would you like me to…
- Here is a more detailed breakdown
- As an AI language model…
- As of my last knowledge update…
- Based on available information…
- While specific details are limited…
- Mad-libs placeholders: [Your Name], [link], INSERT_*, 2025-XX-XX
- Collaboration filler: "Happy to refine this further!"

## Edit summaries and meta (if applicable)

Avoid edit/PR summaries that:

- Stack vague policy assurances ("ensured neutrality, verifiability, and encyclopedic tone")
- Stress what was "preserved/retained" without need
- Lead with "added sourced content" instead of what changed
- Use markdown/emoji in the summary line

Say what changed: "Cut legacy filler from lead; name IUCN status with citation."

## Signs that are fine (do not overcorrect)

- Correct grammar and clear structure
- Occasional transition words
- Formal technical prose without the vocabulary cluster above
- Real challenges discussed with specifics
- Straight-facing "is/are/has" sentences
- Sentence-case headings and plain bullets

## Revision recipe

1. Delete sentences whose only job is importance, legacy, or future outlook.
2. Replace AI vocabulary; accept repetition of plain nouns.
3. Restore "is/are/has" where you had "serves as/boasts/features".
4. Collapse "not only/but also" into one clause.
5. Convert bold-label lists and emoji headers to prose where possible.
6. Remove chatbot openers, closers and hedges.
7. Read aloud: if it sounds like a press release or a slide deck, cut again.
